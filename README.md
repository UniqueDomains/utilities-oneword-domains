## ⚠️ Methodology and caveats

This set focuses on one-word domains connected to utilities terms, especially utility and electricity. The breadth is unusually wide, with 14,922 domains spread across 505 TLDs, so extension quality matters as much as the keyword itself. Sample names show a mix of exact-match utility terms and broader words paired with niche endings, including utility.ryukyu and electricity.hamburg. When comparing these domains, start with the keyword’s commercial clarity, then weigh the TLD for trust, memorability, and renewal fit. For founders, the main question is whether the full name is credible and easy to keep. For investors, the key is whether the keyword-TLD pairing supports realistic resale demand at the current ask.

- Median ask is $1,199 across this utilities-focused selection
- Coverage is broad: 14,922 domains in 505 different TLDs
- Exact utility terms appear alongside electricity keyword variants
- Favor pairings where keyword and extension make sense together
